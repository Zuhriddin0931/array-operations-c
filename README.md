# Array Operations in C

This project is a simple C program that performs basic array operations using a menu system.

## Features
- Print even numbers
- Decrease all numbers by 1
- Find index of a number
- Sort array
- Count occurrences of a number
- Display array


# Операции с массивами на C

Этот проект — простая программа на языке C, которая выполняет основные операции с массивами через меню.

## Возможности
- Вывод чётных чисел
- Уменьшение всех чисел на 1
- Поиск индекса числа
- Сортировка массива
- Подсчёт количества вхождений числа
- Вывод массива
